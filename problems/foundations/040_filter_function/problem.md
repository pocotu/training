# [F040] filter() function

## Problema

Escribe una función llamada `filter_even_numbers` que reciba una lista de números `lst` y devuelva una nueva lista solo con los números pares. Debes usar la función `filter()`.

## Ejemplos

### Ejemplo 1:
```
Input: lst = [1, 2, 3, 4, 5]
Output: [2, 4]
```

### Ejemplo 2:
```
Input: lst = []
Output: []
```

### Ejemplo 3:
```
Input: lst = [1, 3, 5]
Output: []
```

## Constraints

- La función debe llamarse `filter_even_numbers`.
- DEBE usar la función built-in `filter()`.
- El resultado debe ser una lista (convertir de filter object).
- La lista puede estar vacía.
