# [F001] Hello, World!

## Problema

Escribe una función llamada `hello_world` que no reciba ningún argumento y devuelva el string "Hello, World!".

## Ejemplos

### Ejemplo 1:
```
Input: N/A
Output: "Hello, World!"
```

## Constraints

- La función debe llamarse `hello_world`.
- No debe tomar ningún argumento.
