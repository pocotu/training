# [F044] Bubble Sort

## Problema

Escribe una función llamada `bubble_sort` que reciba una lista de números `lst` y devuelva una nueva lista con los elementos ordenados de menor a mayor usando el algoritmo bubble sort.

## Ejemplos

### Ejemplo 1:
```
Input: lst = [64, 34, 25, 12, 22, 11, 90]
Output: [11, 12, 22, 25, 34, 64, 90]
```

### Ejemplo 2:
```
Input: lst = [5, 2, 8, 1, 9]
Output: [1, 2, 5, 8, 9]
```

## Tags
sorting, algorithm, bubble-sort, foundations

## Notas Adicionales
- **Complejidad de tiempo**: O(n²)
- **Complejidad de espacio**: O(1)
- **Conceptos clave**: Algoritmos de ordenamiento, comparaciones, intercambios
